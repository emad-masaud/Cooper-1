The View Transitions API allows you to animate between page states, creating a seamless, app-like feel for your multi-page application (MPA).

### How It Works

Astro automates the View Transitions setup. With a single `<ViewTransitions />` component, you get:
*   **Cross-fade animations**: Smooth transitions between pages.
*   **Persistent elements**: Keep a video playing or a header fixed while navigating.
*   **Scroll restoration**: Maintained scroll position on back navigation.

This fundamentally changes the user experience of static sites, making them feel as fluid as rigorous Single Page Applications (SPAs).
